## Inisialisasi Project

Jalankan perintah berikut di terminal:

```bash
# Buat folder project
mkdir go-api-gateway

# Masuk ke folder project
cd go-api-gateway

# Inisialisasi Go module
go mod init go-api-gateway

# Jalankan Aplikasi
go run ./cmd/gateway
